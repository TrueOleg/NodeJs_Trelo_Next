2) Npm install
    <pre>sudo apt-get install npm</pre>

3) PostgreSQL install
    <pre>https://www.digitalocean.com/community/tutorials/postgresql-ubuntu-16-04-ru</pre>

    <pre>https://www.postgresql.org/download/windows/</pre>

    ------------------
4) Install project
   <pre>npm install</pre>
    ------------------
5) Start project
   <pre>npm start</pre>
